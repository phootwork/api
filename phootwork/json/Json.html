static array decode(string $json, int $options = 0, int $depth = 512)

Decodes a JSON string to an array.

Parameters

string $json The json string being decoded. This only works with UTF-8 encoded strings.
int $options Bitmask consisting of JSON_FORCE_OBJECT, JSON_HEX_QUOT, JSON_HEX_TAG, JSON_HEX_AMP, JSON_HEX_APOS, JSON_INVALID_UTF8_IGNORE, JSON_INVALID_UTF8_SUBSTITUTE, JSON_NUMERIC_CHECK, JSON_PARTIAL_OUTPUT_ON_ERROR, JSON_PRESERVE_ZERO_FRACTION, JSON_PRETTY_PRINT, JSON_UNESCAPED_LINE_TERMINATORS, JSON_UNESCAPED_SLASHES, JSON_UNESCAPED_UNICODE
int $depth User specified recursion depth.

Return Value

array Returns the value encoded in json in appropriate PHP type. Values true, false and null are returned as TRUE, FALSE and NULL respectively.

Exceptions

JsonException if something gone wrong

static string encode(mixed $data, int $options = 0, int $depth = 512)

Returns the JSON representation of a value

Parameters

mixed $data The value being encoded. Can be any type except a resource. All string data must be UTF-8 encoded.
int $options Bitmask consisting of JSON_FORCE_OBJECT, JSON_HEX_QUOT, JSON_HEX_TAG, JSON_HEX_AMP, JSON_HEX_APOS, JSON_INVALID_UTF8_IGNORE, JSON_INVALID_UTF8_SUBSTITUTE, JSON_NUMERIC_CHECK, JSON_PARTIAL_OUTPUT_ON_ERROR, JSON_PRESERVE_ZERO_FRACTION, JSON_PRETTY_PRINT, JSON_UNESCAPED_LINE_TERMINATORS, JSON_UNESCAPED_SLASHES, JSON_UNESCAPED_UNICODE
int $depth Set the maximum depth. Must be greater than zero.

Return Value

string Returns a JSON encoded string

Exceptions

JsonException if something gone wrong
